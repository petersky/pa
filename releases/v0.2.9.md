# v0.2.9 — 0.2.9

**Release track:** release  
**Date:** 2026-07-18

## Summary

This release hardens PA for more reliable autonomous and fleet operation: Codex device login, lease-authority failover, dispatch convergence, and host restart budgets, plus chat Markdown/pagination and restored agent settings.

## Highlights

- Codex device login with more reliable instruction capture and lifecycle timeouts
- Fleet health checking with corrected refresh terminal states
- PR lease authority decoupled from fleet owner, with safer fence handling across replicas
- Autonomous Mac mini host operation hardened (service restart and shutdown budgets)
- Chat renders user Markdown and pagination behaves correctly; agent settings controls restored

## Added

- Codex device login flow
- Fleet health checking
- User Markdown rendering in chat
- Restored agent settings controls

## Changed

- Decple PR lease authority from fleet owner
- Preserve higher fences from stale replicas
- Make fleet dispatch completion converge
- Harden autonomous host operation and honor service restart/shutdown budgets
- Harden batched agent settings apply; keep pending state through refresh

## Fixed

- Queued Codex login output timeout and login lifecycle timeouts
- Codex device login instruction capture
- Fleet health refresh terminal states
- Chat pagination and pagination error status
