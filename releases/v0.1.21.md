# v0.1.21 — 0.1.21

**Release track:** release  
**Date:** 2026-07-16

## Summary

This is a maintainer-focused release that improves the `./scripts/release.sh` workflow. There are no end-user PA behavior changes—only tooling for cutting releases and generating release notes.

## Highlights

- Clearer step-by-step progress output when running `./scripts/release.sh`
- Configurable timeout for the release-notes agent (`--agent-timeout`, `PA_RELEASE_AGENT_TIMEOUT`, default 300s)
- Actionable error message when the notes agent exceeds its time limit

## Added

- `--agent-timeout` CLI flag and `PA_RELEASE_AGENT_TIMEOUT` environment variable for release notes generation

## Changed

- Release workflow now logs each major step (changelog gathering, agent invocation, version bump, GitHub publish) as it runs

## Upgrade

pa update

curl -fsSL https://raw.githubusercontent.com/petersky/pa/main/scripts/install-remote.sh | bash
