# v0.2.6 — 0.2.6

**Release track:** release  
**Date:** 2026-07-18

## Summary

This release adds native fleet instance updates and hardens update coordination so fleet operators can roll out safely with clearer progress and recovery. It also fixes large transcript restoration and SSE catch-up for long-running sessions.

## Highlights

- Native fleet instance updates with endpoint UI
- Idempotent update dispatch and recovery so retries don’t double-fire work
- Live jobs preserved during fleet reconciliation
- Stronger update verification by channel identity
- Large transcript restoration and SSE catch-up fixed

## Added

- Native fleet instance updates (endpoint + UI)

## Changed

- Fleet updates verified by channel identity
- Fleet update coordination, progress reporting, and edge-case handling hardened
- Fleet update recovery made idempotent

## Fixed

- Idempotent update dispatch can resume cleanly after interruption
- Live jobs no longer dropped during reconciliation
- Large transcript restoration and SSE catch-up for long sessions
