# v0.2.15 — 0.2.15

**Release track:** release  
**Date:** 2026-07-22

## Summary

This release makes remote work more durable and automatic: async remote dispatch, recoverable realm-head convergence, first-class repository resources with fenced card worktrees, and centralized PA prompts in Settings. Operators and anyone running PA against remote realms or repositories should update.

## Highlights

- Asynchronous, durable remote dispatch (#54)
- Automatic, recoverable realm-head convergence (#52)
- First-class repository resources with fenced card worktrees (#51, #50)
- PA prompts centralized and editable in Settings (#53)
- Stricter fence mismatch rejection and non-converged push conflicts

## Added

- First-class repository resources (#51)
- Fenced card worktree provisioning (#50)
- Centralized PA prompts exposed in Settings (#53)

## Changed

- Remote dispatch is now asynchronous and durable (#54)
- Realm-head convergence is automatic and recoverable (#52)
- Push conflicts remain non-converged until resolved

## Fixed

- Reject persisted execution fence mismatches
- Keep push conflicts marked non-converged

## Upgrade

```bash
pa update
# or fresh install:
curl -fsSL https://raw.githubusercontent.com/petersky/pa/main/scripts/install-remote.sh | bash
```
