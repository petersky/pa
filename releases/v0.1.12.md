# v0.1.12 — 0.1.12

**Release track:** release  
**Date:** 2026-07-09

## Summary

This release polishes the multi-session agent chat experience introduced in v0.1.11. Users who chat with agents in the browser get session switching, per-card agent panes, and configurable agent options; anyone running `pa stop` or `pa restart` with active ACP sessions should upgrade for a reliable quiesce path.

## Highlights

- Agent page Sessions sidebar with live list, switching, and New session
- Per-card agent chat embed in card detail panes
- Dynamic agent config options in the chat toolbar
- CLI quiesce CSRF priming so `pa stop` and `pa restart` can quiesce sessions without 403 errors

## Added

- Sessions sidebar on the Agent page with a live session list and New session control
- Compact agent chat widget embedded in card detail panes, scoped to `card:{id}` sessions
- Toolbar UI for agent config options (boolean toggles and select controls), persisted via the session config API

## Changed

- Switching sessions reloads the transcript and reconnects SSE without a full page refresh
- Session list stays in sync when sessions are created, closed, or switched

## Fixed

- CLI `pa stop` and `pa restart` prime the CSRF cookie before POST `/api/agent/quiesce`, avoiding 403 failures on localhost
- Pending permission prompts restore correctly when loading a session snapshot

## Upgrade

```bash
pa update
# or fresh install:
curl -fsSL https://raw.githubusercontent.com/petersky/pa/main/scripts/install-remote.sh | bash
```
