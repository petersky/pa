# v0.1.20 — 0.1.20

**Release track:** release  
**Date:** 2026-07-16

## Summary

This is a small stability release for anyone using agent chat in the Web UI. It fixes a crash that could kill the Server-Sent Events (SSE) stream when reconnecting or replaying transcript events.

## Highlights

- Agent chat SSE no longer crashes with `UnboundLocalError` when streaming session events

## Fixed

- Agent chat SSE stream no longer dies on `UnboundLocalError` for `after_seq` when replaying events before subscribe
