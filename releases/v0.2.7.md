## Summary

This release hardens Codex login for multi-process and fleet use, adds durable fleet-wide PR supervision, and makes data-dir and `uv` resolution more reliable during install and update.

## Highlights

- Durable fleet-wide PR supervision with safer review and retirement handling
- Remote Codex device authentication plus cross-process login lease safety
- Codex login recovery and conflict details preserved across the fleet
- `PA_DATA_DIR` honored before config merge
- More robust `uv` resolution for PA updates

## Added
