# v0.2.14 — 0.2.14

**Release track:** release  
**Date:** 2026-07-22

## Summary

This release rebuilds Home and Work around shared card summaries and an action-oriented board UX, and makes restarts and updates shut down cleanly by releasing long-lived SSE streams promptly.

## Highlights

- Shared durable card summaries with provenance, freshness, and an accessible detail dialog
- Home rebuilt as an action-oriented command center; Work as a filterable responsive board
- Graceful shutdown now notifies and drains SSE streams so restarts no longer hang on open EventSource connections
- Quieter production logs and less frequent GitHub credential probing
- CSRF tokens available on first render so embedded card actions work safely

## Added

- Durable card summaries with provenance, freshness state, deterministic fallback generation, and SQLite/event-log migration
- Reusable compact card summaries and an accessible URL-backed card detail dialog with explicit edit and agent start/resume actions
- Accessible Move menus, optimistic drag/drop with rollback, mobile lane switching, and reduced-motion support
- Loading, error, deep-link, and history behavior for core work surfaces

## Changed

- Home is now an action-oriented command center; Work is a filterable responsive board without persistent right rails
- CSRF tokens are available on the first rendered response for embedded card actions
- Uvicorn graceful connection draining is capped at 10 seconds; SSE streams are notified on shutdown
- Successful `httpx` request logs are suppressed; GitHub credential probes run every 15 minutes (errors retry every minute; re-probe on credential change)

## Fixed

- Avoid persisting no-op card edits
- Stop transcript replay during shutdown so streams can close promptly
- Restart/update no longer stalls waiting for browser EventSource connections to close
