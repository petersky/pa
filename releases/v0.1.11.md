# v0.1.11 — 0.1.11

**Release track:** release  
**Date:** 2026-07-09

## Summary

This release brings live agent chat to the PA web UI. Users who work with agents in the browser get a streaming transcript on the Agent page, non-blocking prompts with a message queue, and inline permission handling backed by new REST and SSE APIs.

## Highlights

- Modular Agent page chat widget with live SSE transcript streaming
- REST and SSE APIs for multi-session agent chat
- Non-blocking prompts and queued messages while the agent is working
- Inline permission prompts with ACP-shaped responses and auto-approve preference

## Added

- Agent chat REST/SSE APIs for sessions, prompts, permissions, models, modes, and preferences
- Modular, embeddable `AgentChatWidget` on the Agent page with configurable toolbar options
- Permission auto-approve preference (per-user and global)

## Changed

- Agent turns stream over SSE with a live transcript instead of blocking on HTMX turn completion
- Prompts are non-blocking; the composer clears on send and messages queue while the agent is working

## Fixed

- Permission responses return ACP-shaped outcomes with inline UI and queue controls
