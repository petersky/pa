# v0.1.6 — 0.1.6

**Release track:** release  
**Date:** 2026-07-07

## Summary

This release strengthens PA’s web UI and security foundation: sign-in with CSRF protection, new card and project workflows, and hardening across sync, persistence, cookies, and shared fleet stores. Anyone running the PA web app or multi-agent fleet should upgrade for the auth, sync, and UI fixes.

## Highlights

- Login page with CSRF protection and persistence hardening
- Card edit UI and project creation form
- Fixes for sync integrity, auth enforcement, and critical UI bugs
- Hardened cookies, lane moves, themes, and shared fleet stores

## Added

- Login page
- CSRF protection
- Card edit UI
- Project creation form

## Changed

- Hardened cookies, lane moves, themes, and shared fleet stores
- Persistence hardening for web session and stored state

## Fixed

- Sync integrity issues
- Auth enforcement gaps
- Critical UI bugs
- Agent permission handling
