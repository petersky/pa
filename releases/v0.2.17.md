# v0.2.17 — 0.2.17

**Release track:** release  
**Date:** 2026-07-23

## Summary

This patch release improves PA reliability across multi-instance MCP routing, concurrent persistence, durable session recovery, workspace cleanup, Fleet freshness, and agent session diagnostics. It also refines Fleet and mobile chat usability.

## Highlights

- Route PA MCP requests through the service owned by the selected instance (#76)
- Prevent concurrent atomic JSON writers from colliding (#77)
- Recover durable agent sessions after host interruptions (#75)
- Reconcile completed Fleet workspaces and harden cleanup (#73, #74)
- Add searchable closed-session history and clearer lifecycle diagnostics (#72)

## Changed

- Keep Fleet freshness snapshots consistent and make topology responsive
- Group Fleet topology activity edges for clearer visualization (#67)
- Improve mobile agent chat layout
- Improve agent session lifecycle diagnostics and history (#72)

## Fixed

- Fix owner-scoped PA MCP service routing (#76)
- Fix concurrent atomic JSON writes (#77)
- Recover durable sessions after host interruption (#75)
- Reconcile completed Fleet workspaces and harden workspace cleanup (#73, #74)
- Fix supervisor workspace and transcript shutdown races
- Fix repository form route shadowing
